Funkce 'Print()' a proměnné



Funkce Print()

Používá se k "vytištění" textu na terminál. Text musí být v úvozovkách buď takto ("text") nebo takto ('text')

Příklad:
print("Tento text se vytiskl na konzoli.") >>Tento text se vytiskl na konzoli.

Co je to Proměnná

Proměnná = "šuplík" ve kterém se dají uchovat informace. Nesmí také obsahovat diakritiku

Uložení hodnoty do proměnné + tisk hodnoty na obrazovku

Pokud chceme proměnou vytisknout, napíšeme do funkce print() jméno proměnné bez uvozovek.

Příklad:
 jmeno="Josef"
vek = 53
print(jmeno ,' má ', vek ,' let ')
>>Josef má 53 let

Python má pět standardních datových typů

  • Numbers
  • String
  • List
  • Tuple
  • Dictionary

Python zárověň podporuje čtyři různé číselné typy

  • Int = celá čísla (-1, 0, 1,...)
  • Float = desetinná čísla (0.200, 0.004, 5.000,...)
  • String = text (Hello World!)
  • List = může obsahovat množství dat, zapisuje se v hranatých závorkách ['list', 5, 2.554]

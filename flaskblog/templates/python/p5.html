Tuple a Dictionary



Tuple je neměnný list, k datům se dá dostat, ale nemůžou se měnit.
Tuple se vytváří stejně jako list, ale používají se kulaté závorky.
Vytvoření tuplu: tuple = (1, 'string', 1.5)

Může mít pouze dvě metody:
  • .count() vrátí kolikrát se zadaný prvek v tuplu objevuje
  • .index() vrátí pozici zadaného prvku v tuplu

Dictionary se používají k uchování párů dat ve tvaru: key(klíč):value(hodnota) Dá se měnit, nesmí obsahovat duplikáty Vytvoření dictionary: 
dict = {
    "jmeno": "honza"
    "prijmeni": "novy"
    "vek": 30
}
print(dict) - vypíše celý dictionary
print(dict["jmeno"]) - vypíše pouze hodnotu od klíče jméno

Do dictionary se dají data dodatečně přidat: dict["mesto"] = "Praha"
Užitečné Metody:
  • .clear() - Odebere vše z dictionary
  • .keys() - Vrátí všechny keys z dictionary
  • .copy() - Vrátí kopii dictionary
  • .pop() - Odebere data se zadaným klíčem

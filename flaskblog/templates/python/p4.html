List


Co je v Pyhonu List
List je jedním ze 4 předdefinovaných datových typů v Pythonu používaných k ukládání sbírek dat, další 3 jsou Tuple, Set a Dictionary. Položky List jsou seřazeny, lze je měnit a umožňují duplicitní hodnoty. Položky List jsou indexovány, první položka má index [0], druhá položka má index [1] atd.

List se vytváří pomocí hranatých závorek. obsahuje množství dat, oddělených čárkami.
Vytvoření listu: list = [1, 'string', 1.5]
List může být i také prázdný: list = []

Užitečné metody a příklady:
vytvoříme list muj_list = ["Adam", "Petr", "Petr"]
vytiskneme list print(muj_list)
  • .index vytiskneme jméno pomocí indexu takto :muj_list.index(0) vytiskne se jméno Adam

  • .append přidáme prvek do listu takto : muj_list.append("Martin") když list vytiskeneme pomocí print(muj_list) vytiskne se ["Adam", "Petr", "Petr","Martin"]

  • .insert přidáme prvek na pozici podle indexu do listu takto : muj_list.insert(0,"Josef") když list vytiskeneme pomocí print(muj_list) vytiskne se ["Josef","Adam", "Petr", "Petr","Martin"]

  • .clear() vymažeme vše v Listu takto : muj_list.clear() Vymaže všechny prvky z listu

  • .sort() seřadímě položky v listu podle abecedy nebo hodnoty : muj_list.sort()

  • .reverse() položky v listu seřadíme obráceně muj_list.sort()
další užitečné metody funkce

  • .extend chceme-li k aktuálnímu listu přidat prvky z jiného listu

  • .muj_list[1] = "Karel" chceme-li změnit hodnotu položky v indexu 1

  • .extend() položku přidáme na konec listu

  • .remove("Adam") odstraníme poožku "Adam" z listu

  • .copy() vytvoříme kopii listu muj_novy_list = muj_list.copy()

  • .count("Petr") vypíšeme počet výskytů hodnoty "Petr" v listu

Manipulace s obrázkem s pillow


Pillow nainstalujeme v terminálu pip install pillow

Spuštěním tohoto kódu zobrazíme image
from PIL import Image

image = Image.open('foto.jpg')
image.show()

Vytvoříme kopii
image = Image.open('foto.jpg')
image.save('foto.png')

Vypíšeme všechny soubory s příponou .png
from PIL import Image
import os

for f in os.listdir('.'):
  if f.endswith('.png'):
  print(f)

Vypíšeme názvy(připony) všech souboru .png
for f in os.listdir('.'):
   if f.endswith('.png'):
       i = Image.open(f)
       nazev,pripona = os.path.splitext(f)
       print(nazev)

Zkopírujeme všechny soubory s příponou .png do předem vytovořené složky /pngs
for f in os.listdir('.'):
   if f.endswith('.png'):
       i = Image.open(f)
       nazev,pripona = os.path.splitext(f)
       i.save('pngs/{}.png'.format(nazev))

Přejmenujete a vytvoříte kopie souborů s koncovkou .png do předem vytovořené složky /300 v rozlišení 300x300px
size_300 = (300,300)

for f in os.listdir('.'):
   if f.endswith('.png'):
       i = Image.open(f)
       nazev,pripona = os.path.splitext(f)
       i.thumbnail(size_300)
       i.save('300/{}_300.{}'.format(nazev,pripona)))

Otočíte obrázek a uložite jako nový soubor
image = Image.open('foto.jpg')
image.rotate(90).save('otocenene_foto.jpg')

Vytvoříte černobílou kopii a uložite jako nový soubor
image = Image.open('foto.jpg')
image.convert(mode=´L').save('cernobila_foto.jpg')

Vytvoříte rozmazanou kopii a uložite jako nový soubor
image = Image.open('foto.jpg')
image.filter(ImageFilter.GaussianBlur(15)).save('rozmazana_foto.jpg')

V tomto odkazu najdete kompletní příručku pillow

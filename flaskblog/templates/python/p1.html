Instalace Python



1.krok Stáhněte si nejnovější verzi Pythonu pro váš systém na stránce: https://www.python.org/downloads/

2.krok Vyberte si textový editor ve kterém chcete programovat. Můžete využít jakýkoliv textový editor jako je např. poznámkový blok a atd. Samozřejmě existují i speciální editory pro psaní python-kódu tzn IDE(integrated development environment ), neboli integrované vývojové prostředí. Doporučuji si stáhnout jeden z těchto editorů.

  • PYCHARM
  • VISUAL STUDIO CODE
  • ATOM
  • SUBLIME TEXT

V případě, že si stáhnete PYCHARM, aplikace vás po spuštění vyzve k vytvoření nového projektu "New project", vytvořte si ho tedy a nazvěte například "helloworld". V tomto projektu si vytvořte soubor a pojmenujte ho třeba "app.py".
(soubor vytvoříte klinkutím pravým tlačítkem v menu na "hello world" new -> python file,viz obrázek)










V případě, že si stáhnete jiný editor stačí, když si třeba na ploše vytvoříte složku "helloworld" a v ní si vytvoříte soubor "app.py"
Poté můžete spustit svůj program v příkazovém řáadku pomocí příkazu python app.py

Další možnost pro jednoduché programy je využítí on line python editorů.

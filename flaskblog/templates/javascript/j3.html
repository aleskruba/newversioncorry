JavaScript - Editace textu a Proměnné

Editace textu vypadá takto :

document.write("ahoj lidi");

Před použitím proměnné je nejprve nutné ji deklarovat. Deklarace proměnné vypadá takto :
var mesto = "Brno";
var cislo = 15;
var nadpis = "<H1>>Ahoj všichni</H1>";

Zápis proměnné vypadá takto :
var mesto = "Brno ";
document.write(mesto);
pokud chcete provést zápis do consoly použijte: console.log(mesto);
poznámka: Název proměnné nesmí být uvnitř uvozovek, jinak by prohlížeč nepoznal, že jde o proměnnou

Vyzkoušejte si to

<html> <title> JavaScript - Proměnné </title> <head> <script> </script> </head> <body> </body> </html>

Můžete kombinovat text s proměnnou takto
var mesto = "Brno ";
document.write("Mé oblíbené město je " + mesto);

Pokud chcete ve vašem textu uvést uvozovky je nutné provést zápis následovně "
document.write("takto napíšu \' slovo s uvozovky \' uvnitř textu ")

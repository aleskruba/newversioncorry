JavaScript DOM

Co je DOM v JavaScriptu? anglicky Document Object Model (DOM).
JavaScript může mít přistup ke všem prvkům(div,ul,li atd...) webové stránky s využitím modelu (DOM)

Jak používat DOM?
Pomocí DOM může JavaScript provádět více úkolů. Může vytvářet nové prvky a atributy, měnit stávající prvky a atributy a dokonce odstraňovat existující prvky a atributy. JavaScript může také reagovat na existující události a vytvářet nové události na stránce.


getElementById, innerHTML - příklad

  • getElementById: Přístup k prvkům a atributům, jejichž ID je nastaveno.
  • innerHTML: Pro přístup k obsahu prvku.


<html>
<head>
<title> JavaScript - DOM </title>
</head>
<body>
<h1 id="zprava">Ahoj</h1>
<p>Toto je uvítací zpráva </p>
<h2>Obsah</h2>
<p>Toto je obsah zprávy.</p>
<script type="text/javascript">
   var text = document.getElementById("zprava").innerHTML;
   alert("první nadpis je " + text);
</script>
</body>
</html>

getElementsByTagName - příklad

<html>
<head>
<title>DOM</title>
</head>
<body>
<h1>Ahoj</h1>
<p>Toto je uvítací zpráva .</p>
<h2>Obsah</h2>
<p id="dva">Toto je obsah druhé zprávy.</p>
<script type="text/javascript">
  var odstavec = document.getElementsByTagName("p");
   alert("Obsah v druhém odstavci je " + odstavec[1].innerHTML);
   document.getElementById("dva").innerHTML = "Původní druhá zpráva byla změněna na tuto zprávu.";
</script>
<body>
</html>

JavaScript - cykly for, while , do while

Jak používat cykly: Cykly (anglicky loops) jsou užitečné, když musíte opakovaně spouštět stejné řádky kódu. Předpokládejme, že na své webové stránce chcete stokrát napsat větu „Dobrý den“. Místo zkopírování 100x stejného ködu, použijete cykly a celý kód bude mít maximálne 4 řádky.

cyklus for

var jmena = new Array("Adam", "Petr", "Pavel");
for (i=0;i < jmena.length; i++)
{ document.write(jmena[i] + "<br />"); }

    vysvětlení výroků v cyklu for (i=0;i < jmena.length; i++)
  • i=0; je vykonáno ještě před provedením smyčkového kódu. Tento výrok se tedy běžně používá k přiřazení hodnot k proměnným, které se použijí uvnitř cyklu
  • i < jmena.length; tento výrok je podmínkou pro provedení cyklu.
  • i++tento výrok se provede pokaždé po provedení kódu cyklu.

cyklus while
Cyklus "while" je vykonáván, dokud je splněna zadaná podmínka.

V cyklu "while" byste měli zahrnout příkaz, který smyčku v určitém okamžiku ukončí. Jinak tento "cyklus" nikdy neskončí a váš prohlížeč může selhat. Pokud budete psát tento kód zde, zadejte "while (x<20)" až naposledy.

var x = 1;
while (x < 20) {
x++; }
document.write("x < 20");


cyklus do while
Cyklus "do while" je velmi podobný cyklu "while". Jediný rozdíl je v tom, že v cyklu "do while" se blok kódu provede jednou ještě před kontrolou stavu.

V cyklu "do while" byste měli zahrnout příkaz, který smyčku v určitém okamžiku ukončí. Jinak tento "cyklus" nikdy neskončí a váš prohlížeč může selhat.

var x = 1;
do { document.write(x+" ");
x++; }
while(x<10);


Příklad násobky 2
var x = 1;
do { document.write(2*x +",");
x++; }
while(x<=10);

JavaScript podmínky - IF, Else, Else IF

Jak používat podmínky: Podmínky se používají k rozhodnutí o postupu provádění na základě různých podmínek. Pokud je podmínka pravdivá, můžete provést jednu akci a pokud je podmínka nepravdivá, můžete provést další akci

podmínka IF

var vek = prompt("zadej svůj věk ");
if (vek>18)
document.write("je ti více než osmnáct");
if (vek<18)
document.write("je ti méně než osmnáct");

podmínka IF...ELSE
var cas = new Date().getHours();
if(cas<12)
document.write("Dobré ráno");
else
document.write("Dobré odpoledne");


podmínka ELSE IF
var x = prompt("zadej první číslo");
var y = prompt("zadej druhé číslo");
x = parseInt(x);
y = parseInt(y);
if (x == y)
document.write(x + " se rovná " + y );
else if (x<y)
document.write(x + " je menší než " + y );
else
document.write(x + " je větší než " +y );

*Funkce parseInt() převede datový typ "string" na "integer".

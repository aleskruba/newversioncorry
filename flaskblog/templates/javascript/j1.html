Co je JavaScript

JavaScript je open source a nejpopulárnější skriptovací jazyk podporovaný všemi prohlížeči. JavaScript se používá hlavně k posílení interakce uživatele s webovou stránkou. Jinými slovy, pomocí JavaScriptu můžete svou webovou stránku oživit a učinit více interaktivní. JavaScript se také hojně používá při vývoji her a vývoji mobilních aplikací.

JavaScript, jakožto je skriptovací jazyk, nelze spustit samostatně. Spuštění kódu JavaScript se provádí v prohlížeči. Hlavní výhodou JavaScriptu je, že všechny moderní webové prohlížeče JavaScript podporují. Nemusíte si dělat starosti s tím, zda návštěvník vašeho webu používá Internet Explorer, Google Chrome, Firefox nebo jakýkoli jiný prohlížeč. JavaScript bude podporován. JavaScript také běží na jakémkoli operačním systému, včetně Windows, Linux nebo Mac.

Celý kód JavaScriptu můžete umístit do značek <script> </script> nebo si vytvořit kód do samostatného souboru například main.js , (důležitá je přípona .js) poté vložit do HTML souboru následovně: <script src="myscripts.js"> </script>

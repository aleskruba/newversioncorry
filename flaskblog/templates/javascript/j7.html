JavaScript funkce a volání funkce

Co je funkce v JavaScriptu? Funkce jsou velmi důležité a užitečné v jakémkoli programovacím jazyce, protože umožňují opětovné použití kódu. Funkce je blok kódu, který se provede, pouze pokud je vyvolán. Pokud máte několik řádků kódu, které je třeba použít několikrát, můžete vytvořit funkci včetně opakujících se řádků kódu a poté funkci volat kdekoli chcete.

Jak vytvořit funkci v JavaScriptu

  1. Napište keyword "function" následovanou názvem funkce.
  2. Za názvem funkce otevřete a zavřete závorky.
  3. Po závorkách otevřete a zavřete složené závorky.
  4. Do složených závorek napište řádky kódu.
function mojefunkce()
{
document.write("Toto je moje první funkce.");
}
mojefunkce();
